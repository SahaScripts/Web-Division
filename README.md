# Multi-Page Website Using HTML and CSS

## Project Overview
This project is a simple multi-page website developed using HTML and CSS. It demonstrates the use of divisions (`div`), basic page layout, navigation between multiple pages, and styling using CSS. The website contains a Home page with multiple content divisions, along with separate About and Contact pages.

## Pages Description

### 1. Home Page (index.html)
The Home page is the main page of the website. It contains:
- A navigation bar with links to Home, About, and Contact pages
- A welcome message:  
  **“Hello from Arnadeep Saha, welcome to my homepage”**
- Three horizontal content divisions displayed one per row

Each division contains details related to the Web Programming course, including:
- Course name
- Theory and lab schedule
- Faculty name

The divisions are styled using different background colors to visually separate the content.

### 2. About Page (about.html)
The About page displays personal details of the student.  
It includes:

- Branch: CSE Core  

This page is accessed by clicking the **About** link in the navigation bar and opens in a new page.

### 3. Contact Page (contact.html)
The Contact page displays basic contact information of the student.  
It includes:
- Phone Number: 8100125230  
- Email ID: sahaarnadeep@gmail.com  

This page is accessed by clicking the **Contact** link in the navigation bar and opens in a new page.

## Features
- Multi-page website navigation using anchor tags
- Use of `div` elements to organize content
- Horizontal layout with one division per row
- Separate pages for About and Contact information
- Consistent color theme and simple design
- No JavaScript used; implemented using pure HTML and CSS

## Technologies Used
- HTML for structure
- CSS for styling and layout


## Purpose of the Project
This project is created for academic practice to understand:
- Basic HTML page structure
- Use of divisions (`div`)
- Page navigation using anchor tags
- Styling web pages using CSS
- Creating and managing a simple multi-page website
